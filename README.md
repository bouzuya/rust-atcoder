# rust-atcoder

[AtCoder](https://atcoder.jp/)

## Note

```
$ # See cargo-atcoder README
$ # https://github.com/tanakh/cargo-atcoder
$ contest_name=...
$ cargo atcoder new ${contest_name}
$ cd ${contest_name}
$ # edit src/bin/a.rs
$ cargo atcoder test a # optional
$ cargo atcoder submit a
```
